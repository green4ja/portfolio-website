# portfolio-website

## Description
This is my personal portfolio website showcasing my projects, skills, and experience. It is built using modern web technologies and designed to provide a comprehensive overview of my professional background.

## Features
- Responsive design
- Interactive project gallery
- Contact form
- Blog section

## Technologies Used
- HTML
- CSS

## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/portfolio-website.git
    ```
2. Navigate to the project directory:
    ```bash
    cd portfolio-website
    ```

## Usage
Open your web browser and go to `http://localhost:3000` to view the website.
